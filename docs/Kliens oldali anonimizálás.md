# Folyamatosan érkező adatok kliens oldali anonimizálása

## Alapkoncepció
A legtöbb ma létező anonimizálási megoldás a keletkező adatokat eredeti formájukban, anonimizálatlan formában gyűjti egy központi szerveren, ahol bizonyos idő elteltével vagy bizonyos mennyiségű adat összegyűjtése után végzik el az anonimizálást.
